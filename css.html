üß© The Main Parts of a CSS Rule

selector {
  property: value;
}
  • Selector: Defines which HTML element(s) to style (e.g., p, h1, .class).
  • Property: The aspect of the element you want to change (e.g., color, margin, font-size).
  • Value: The specific setting for that property (e.g., red, 20px, center).
