What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the World Wide Web. HTML describes how web pages are displayed by using elements and tags.

Who is the Founder of HTML?

HTML was created by Tim Berners-Lee in 1991 while working at CERN (European Organization for Nuclear Research). He also invented the first web browser and server.

When did HTML Start?

The first version of HTML was released in 1993. Since then, several versions have been introduced, with HTML5 being the latest major standard.

Basic HTML Syntax

<!DOCTYPE html>
<html>
  <head>
    <title>My First HTML Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>
Output:

Hello World

How to Input Commands

You can use any text editor such as Notepad, VS Code, or Sublime Text to write HTML code. Save your file with the .html extension and open it in any browser to see the result.

HTML Page Structure

An HTML page is divided into two main parts: the <head> and the <body>. The head contains meta information, while the body contains all the visible content of the webpage.

Common HTML Tags

  • <h1> - <h6> : Headings
  • <p> : Paragraph
  • <a> : Hyperlink
  • <img> : Image
  • <div> : Division or container
  • <span> : Inline text container

Attributes in HTML

Attributes give extra information to HTML elements. Example:

<a href="https://www.google.com">Visit Google</a>

Lists

HTML supports ordered and unordered lists:

<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

Tables

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Eljay</td>
    <td>20</td>
  </tr>
</table>

Forms

<form>
  Name: <input type="text"><br>
  Email: <input type="email"><br>
  <input type="submit" value="Submit">
</form>

Semantic Tags

HTML5 introduced semantic tags like <header>, <nav>, <section>, <article>, and <footer>. They improve readability and SEO.

Multimedia

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

<video width="320" height="240" controls>
  <source src="video.mp4" type="video/mp4">
</video>

HTML5 Canvas Example

<canvas id="myCanvas" width="200" height="100" style="border:1px solid black;"></canvas>

Best Practices

  • Use <!DOCTYPE html> at the top.
  • Properly nest elements.
  • Use lowercase for tags and attributes.
  • Always include alt text in images.
